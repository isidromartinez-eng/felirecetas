## El proyecto
Página web de pastelería artesanal "Feli Recetas", La Plata, Argentina.
- Público objetivo: mujeres 25-45 años que valoran lo artesanal y casero
- Stack: HTML y CSS puro (sin frameworks)
- El usuario es principiante — explicaciones simples, código claro

## Archivos principales
- `index.html` → página principal (desplegada en Vercel)
- `configurar.html` → configurador de tortas
- `design_bundle/feli/` → archivos de diseño de referencia
